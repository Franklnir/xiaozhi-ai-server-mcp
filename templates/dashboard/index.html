MCP Manager
📖 Cara Mendapatkan MCP Endpoint dari Xiaozhi.me
  1. Login ke xiaozhi.me → masuk ke Console / Dashboard
  2. Buka konfigurasi Smart Agent (智能体) yang ingin dihubungkan
  3. Di halaman Role Config (角色配置), lihat bagian kanan bawah
  4. Copy URL MCP Endpoint — formatnya: wss://api.xiaozhi.me/mcp/?token=XXXX
  5. Paste URL tersebut ke Token Manager di bawah, lalu Test → Simpan
⚠️ Setiap smart agent punya MCP endpoint sendiri (token unik). Jika token berubah di xiaozhi.me, update juga di sini.
